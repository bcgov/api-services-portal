# Status Banner

The status banner component relays status-related information to the user.

Each story showcases one of the four available statuses (`info` (default), `success`, `pending`, and `error`),
as well as two custom variants (`status` (default), and `outline`).

```javascript
import StatusBanner from '@src/components/StatusBanner';

const MyComponent = () => (
  <StatusBanner title="Title here" status="success" variant="outline">
    Status content here.
  </StatusBanner>
);
```
