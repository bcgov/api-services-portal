# Actions Menu

The `ActionsMenu` is a convenient wrapper around the Charka Menu, ideal for use in list-based data displays.

To use 

```javascript
import { MenuItem } from '@chakra-ui/react';
import ActionsMenu from '@src/components/actions-menu';

const MyComponent = () => (
  <ActionsMenu>
    <MenuItem onClick={() => /* handle click */}>Delete</MenuItem>
  </ActionsMenu>
);
```
